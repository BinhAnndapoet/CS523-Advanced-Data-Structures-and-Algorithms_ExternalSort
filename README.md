# 🟣 AlgoViz - Trình Minh Họa Sắp Xếp Ngoại (External Sort)

**AlgoViz** là một công cụ tương tác mạnh mẽ giúp trực quan hóa thuật toán **External Merge Sort** - kỹ thuật sắp xếp các tập dữ liệu khổng lồ vượt quá dung lượng RAM của máy tính.
